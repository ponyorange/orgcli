* youdao
```
快速中英翻译
使用：youdao [中文/英文 单词或句子]
如：youdao i love you
┌────────────┬──────────┐
│ i love you │ 我爱你。  │
└────────────┴──────────┘
如：youdao 我爱你
┌────────┬────────────┐
│ 我爱你 │ I love you  │
└────────┴────────────┘
作为一个合格的程序员，快速中英翻译太重要了，如遇翻译专业术语：
如：youdao component
┌───────────┬─────────────────────────┬─────────────────────┐
│ component │ n. 组成部份，成分，部件     │ adj. 组成的，构成的   │
└───────────┴─────────────────────────┴─────────────────────┘

无需打开翻译软件，直接编辑器命令行走一波即可，优雅，实在太优雅了
```
